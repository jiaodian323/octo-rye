## Octo-Rye

Octo-Rye是一款包含服务器端和客户端两部分的安卓应用开源工程，该应用是依据用户个人兴趣推荐算法，每日向用户推荐精品文章、热点新闻推送、热门音乐欣赏、图书售卖等功能，该项目还处在早期探索开发阶段。
